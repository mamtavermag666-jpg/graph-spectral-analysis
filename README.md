# Graph Spectral Analysis

This project demonstrates basic spectral graph theory concepts using Python.

## Features
- Graph creation using NetworkX
- Adjacency matrix computation
- Degree matrix computation
- Laplacian matrix computation
- Eigenvalue analysis
- Graph visualization

## Mathematical Background

The Laplacian matrix is computed as

L = D - A

where:
- D = Degree matrix
- A = Adjacency matrix

The eigenvalues of the Laplacian matrix provide important structural information about graphs.

## Technologies Used
- Python
- NumPy
- NetworkX
- Matplotlib
